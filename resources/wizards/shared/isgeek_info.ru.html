1. Нажмите кнопку «Открыть веб-страницу», чтобы открыть страницу регистрации в веб-браузере.
  2. Введите в поле «JID» желаемое имя пользователя.
  3. Введите желаемый пароль в поле «Heslo».
  4. Введите пароль ещё раз в поле «Heslo (znovu)».
  5. Введите текст с картинки в поле ввода под картинкой.
  6. Нажмите кнопку «Zřídit účet» для продолжения регистрации.
  7. Еcли регистрация прошла успешно, зкройте окно браузера. В противном случае, попробуйте другое имя пользователя.
  8. Нажмите кнопку «Далее» чтобы перейти к странице подключения.
