1. Press «Open registration website» to open registration page in your system web browser.
  2. Type desired user name into «JID» field.
  3. Type desired password into «Heslo» field.
  4. Retype your password into «Heslo (znovu)» field.
  5. Enter text from the picture into the field under the picture.
  6. Press «Zřídit účet» button to proceed registration.
  7. If registatration is successful, close your web browser, or try different user name otherwise.
  8. Press «Next» button below to proceed to connecion page.
