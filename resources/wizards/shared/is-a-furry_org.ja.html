1. Press «Open registration website» to open registration page in your system web browser.
  2. Type desired user name into «Username:» field.
  3. Type desired password into «Password:» field.
  4. Solve the problem after «What is» below and type the answer into field next to it.
  5. Press «Register!» button to proceed registration.
  6. If registatration is successful, close your web browser, or try different user name otherwise.
  7. Press «Next» button below to proceed to credentials page.
