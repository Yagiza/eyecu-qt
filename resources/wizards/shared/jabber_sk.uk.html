1. Press «Open registration website» to open registration page in your system web browser.
  2. If your browser reports about security problem, just ignore it and tel your browser to trust a certificate from this site.
  3. Type desired user name into «Používateľské meno:» field.
  4. Type desired password (6 characters minimum) into «Heslo:» field.
  5. Retype your password into «Potvrdenie hesla:» field.
  6. Press «Vytvoriť» button to proceed registration.
  7. If registatration is successful, close your web browser, or try different user name otherwise.
  8. Press «Next» button below to proceed to credentials page.
