1. Нажмите кнопку «Открыть вебсайт регистрации», чтобы открыть страницу регистрации в веб-браузере.
  2. Введите в поле «Username:» желаемое имя пользователя.
  3. Введите желаемый пароль в поле «Password:».
  4. Решите задачу после «What is» ниже и введите ответ в поле по соседству.
  5. Нажмите кнопку «Register!» для продолжения регистрации.
  6. Еcли регистрация прошла успешно, зкройте окно браузера. В противном случае, попробуйте другое имя пользователя.
  7. Нажмите кнопку «Далее» внизу, чтобы перейти к странице ввода регистрационных данных.
