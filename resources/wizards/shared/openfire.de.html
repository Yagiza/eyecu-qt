1. Press «Open registration website» to open registration page in your system web browser.
  2. Type desired user name into «Username» field.
  3. Type desired password into «Password» field.
  4. Retype your password into «Confirm Password» field.
  5. Optionally, you may enter your real name and e-mail into «Name» and «Email» fields respectively. Those fields will be used for user search.
  6. Press «Create Account» button to proceed registration.
  7. If registatration is successful, close your web browser, or try different user name otherwise.
  8. Press «Next» button below to proceed to credentials page.
